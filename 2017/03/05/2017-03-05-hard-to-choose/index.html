选择困难症

发表于2017-03-05|更新于2025-05-17|工作
|浏览量:

在选择面前，我更喜欢年少时的我：果断、勇敢。在很早的时候，我就被教会：天下是没有后悔药吃的。但每次作出选择后，无论结果如何，我都能找到一个完说的理由：这样的选择也不错吧。直到不久前，前前公司的一场闹剧，开始了自己对人生的怀疑。

到目前为止，我只服务过3家公司，顺丰科技、集货优运和舜飞科技。从顺丰出来回到广州的经历是当时一个无奈的决定。我很喜欢那份工作，但，同时也对自己的待遇感到悲观。远离深圳的好友，独自来到广州，看似转业加薪，实则是无奈的选择。碌碌庸庸在集货优运度过了1年多，经历了创业合伙人的资产纠纷，虽然口声说与产品技术无关，但创业失败的结果又岂能不负上责任呢。匆匆十月得到创业失败的消息，十一月就拿到两个offer了。列了十项比较，并最终选择了舜飞科技。当时自己列的几行优势是这样的：

  1. 数据环境一流，具有挑战性
  2. 自主选择开发方案
  3. 工作内容：语言不限，开发新的平台
  4. 长远：高并发，分布式，数据分析，前后端全栈

可是进去三个月后，发现以上没有一项能达到预期。原因很简单，项目中止了。从那个项目中止后，我突然发现自己处于一个十分尴尬的位置——前端部的后端工程师，再加上前端部与后端部不怎么和谐，连向往长远的目标都尴尬。最后转正聊天时，我已经做好离开的心里准备了——面对自己的失败，需要诚实勇敢地面对。在这个时候，我又回想起顺丰上司的告诫：不要丢我们的脸。想了想，是没有脸见他了。

回归现实，我最近又得对职业作出选择。时值三月，各路招聘旺季。庆幸的是，我的简历在不用投递的情况下，也能招来众多公司的青眯。不过自己投递的两家都没有回应，得总结下原因。在之前的两周内，我一共参与了13场面试，有复试的是5家，最后给了offer的有三家。这是我第二次做出这样的决定。由于失败的阴影，导致我在决定之前思考更多的问题：如何去了解一家的公司前景？如何找到合适自己的发展方向？甚至去需求咨询：一家公司是如何选择合适的人？得到的回答很棒：

  • 现在想要的东西和公司现在能给的东西匹配；
  • 未来想要的和公司未来能提供的接近；
  • 公司定位的这个角色跟他想成为的角色一致。

很多道理在心中都是知道的，但实际去沟通去观察时，往往会忽略各种各样的问题。有些问题是致命的，产生的情绪无法估量。现实迫使我困难地谨慎地选择下一个去处，因为我不希望自己再消极，我的未来必须是个积极的人生。

人生终究是由无数的选择组成，我并不能保证这是我最后一次决定人生的去向。每一次做出选择都要花费大量的时间与精力，力求无悔。但不可避免地总会让自己很不爽，无法原谅的时候。这样，除了时间、精力，更受影响的是情绪化的心。无论如何，在文章的末尾我都会作出选择，剩下的，就是努力了。
