总之是变了

发表于2017-03-26|更新于2025-05-17|活动
|浏览量:

昨晚又刷了一次五月天的演唱会，这是从7年前意外入坑后第三次刷他们的演唱会。在朋友圈上久违地发了一发动态，当然是收到了朋友们的祝福。不过，最为感触的是，@iROCKRUNNY的吐槽。渠小哥在7年前带我去刷Mayday演唱会，从北三环骑自行车到东三环工体的场景还记忆犹新。小哥说他不会去听他们的演唱会了，Mayday的新歌把他们毁了。圈子另一边的我先是愕然了一下，很快我就进入了思考模式。

在我心目中，渠小哥是个很有原则的人，他的人生也是我所向往的。自由的选择深造，欧洲的学习与游历，各种各样的音乐和摄影爱好。他说的不喜欢，肯定有他的道理。我只能说一句，五月天的确是变了。5年中他们没有发过专辑，五年之后的他们在诉说他们的自传。嗯，他们在总结。说句不好听的是，他们在卖老了。但我只会说，我会买这个老。原因很简单，他们在我最无助的时候带给我感动。

总之他们是变了，但我仍然会喜欢，因为我也在变。不忘初心的渠小哥会在喜欢的Rock上继续前行，而我则是在泥泞中翻滚，在生活和生存之间折腾。我觉得我会如大多数的凡人一样，终日奔波苦，一刻不得闲。久而久之，发现很多初心爱好消失了很多。譬如动画，小学开始的爱好，在几个月来两度选择工作的忙碌中几乎消散得无影无踪，B站也很少上了。当你发现即使缺少了也没关系的时候，那就说明你变了。再如卡游老祖宗《游戏王》，从小桥底与发小玩到发烧，到追到最近的钟摆召唤，而现在发小发来游戏王的新规则，我已经当成是茶余饭后的新闻而已。说不定某天，五月天、怪物猎人和海贼王也会成为曾经的怀念？

在这我又得提起《最终幻想15》，这部花了10年磨一剑的游戏。很多人搞完之后都觉得坑，包括我在内其实也觉得不值得要做10年。但把剧情与这10年SE社的折腾联系在一起，又不得不感叹游戏开发之不易：为了取得力量还得逝去10年的光阴和生命？SE社这个隐喻太明白了，纵使他们知道这个团队不容易，走了很多路，中间还有得失离合，但他们还是给玩家最好的东西。嗯，《最终幻想15》团队也好，SE社也好，他们都在变化。玩家们呐喊着，请还我《最终幻想7》时，是否有想过现在的SE社已经不是当时了呢？他们只是一家公司，没责任对玩家们承诺任何，他们在尽自己能力履行自己的使命。

初心又好，使命也罢，总之是变了。阻止变化是最坏的选择。物竞天择，适者生存。要么提出新的想法，譬如我不喜欢了，要么尝试去命中新事物的痛点产生兴趣，进而奋力争取。从五月天中找到那份烦杂中的情感，从怪猎中找到成功讨伐的喜悦，从海贼王中找到一路向前的热血，还有很多很多兴趣爱好填充着自己。五月天在变老，我们也在变老；怪物猎人加入了新招式，我们的工作技术面更广更宽；海贼王越来越多套路，我们又何尝不是被一波三折的套路所感动？
